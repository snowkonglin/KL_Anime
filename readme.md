# KL的基于bangumi/obsidian的动漫库
库中动漫均基于个人喜好保存，欢迎品尝

可直接打包下载到本地并使用obsidian打开
感谢B站UP @鬼头明里单推人 @月涟_luvian 提供的模板及建库思路

## 使用说明

`添加动漫数据.py` 通过bgm提供的api进行生成动漫文档及相关图片信息
功能：创建动漫的简介md及相关图片

示例指令：
`python3 添加动漫数据.py  ”约会大作战“ ` 

> 受限于obsidian的路径读取模式，需要在此项目根目录下将图片目录软链接
示例指令：`ln -s 我的动漫/主页/characters . `
